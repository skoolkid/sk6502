$0EC2: Restart the command list unless it's time to start the lesson
This corresponds to $7C3F in the ZX Spectrum version.
Used by command lists $80, $82, $84, $86 and $B8.
$0EC2 LDA $40 Pick up the MSB of the lesson clock from $40.
$0EC4 CMP #$0B Is it still 11 or greater?
$0EC6 BCS $0ECB Branch if so.
$0EC8 JMP $1D2D Otherwise terminate this primary command.
$0ECB JMP $0EB4 Restart the command list.
