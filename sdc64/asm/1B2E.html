$1B2E: Determine the next move of a character following another character
This corresponds to $7ADC in the ZX Spectrum version.
Called from the main loop at $0800, and also used by the routine at $0C3F. Returns with one of the following values in A depending on the relative locations of the follower and his target:
A Meaning
0 Follower is at the same coordinates as the target
1 Follower should go (or continue going) upstairs
2 Follower should go (or continue going) downstairs
3 Follower should go left
4 Follower should go right
Input
($4D) Number of the target character being followed
